# Roadmap

## v1.0.0 — Initial Release

- [x] Core download engine using yt-dlp
- [x] Video and audio downloading
- [x] Quality and format selection
- [x] Playlist support
- [x] Subtitle support
- [x] Thumbnail downloading
- [x] Metadata extraction
- [x] Resume support
- [x] Batch downloads
- [x] Download queue
- [x] Cookie and proxy support
- [x] Configuration persistence
- [x] Download history
- [x] yt-dlp update command
- [x] FFmpeg detection
- [x] Rich terminal UI

## v1.1.0 — Enhancements

- [ ] Concurrent downloads
- [ ] Download speed limit
- [ ] Playlist item range selection
- [ ] SponsorBlock integration
- [ ] aria2c external downloader support
- [ ] YouTube authentication via OAuth

## v1.2.0 — Advanced Features

- [ ] GUI mode (Tkinter or Web UI)
- [ ] Browser extension companion
- [ ] Watch history sync
- [ ] Download scheduling
- [ ] RSS feed monitoring
- [ ] Plugin system for custom extractors

## v2.0.0 — Major Release

- [ ] Native GUI application
- [ ] Mobile companion app
- [ ] Cloud download support
- [ ] Advanced media management
- [ ] Library organization features
