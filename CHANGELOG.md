# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-07-06

### Added

- Initial release of NovaDL
- Video downloading from all yt-dlp supported sites
- Audio-only extraction (MP3, M4A, Opus, FLAC, WAV)
- Quality selection for video and audio
- Format/extension selection
- Playlist and channel downloading
- Subtitle downloading and embedding
- Thumbnail downloading
- Metadata extraction and display
- Resume interrupted downloads
- Batch URL downloading from file
- Download queue management
- Cookie file support
- Proxy support
- Custom output directory
- Persistent configuration
- Download history tracking
- yt-dlp self-update command
- FFmpeg detection and installation guidance
- Rich terminal UI with progress bars
